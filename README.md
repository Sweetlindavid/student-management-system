# Student Management System

## 📖 About the Project
This is a Full Stack Student Management System developed using React.js, Spring Boot, and MySQL. The application allows users to manage student records efficiently through a modern web interface and REST APIs.

## 🚀 Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- Java
- REST API

### Database
- MySQL

## ✨ Features
- Add Student
- View Students
- Update Student Details
- Delete Student
- Responsive User Interface
- REST API Integration

## 📂 Project Structure

```
student-management-system/
├── frontend/
└── backend/
```

## ▶️ How to Run

### Frontend
1. Open the `frontend` folder.
2. Run:
```
npm install
npm start
```

### Backend
1. Open the `backend` folder.
2. Configure MySQL database.
3. Run the Spring Boot application.
